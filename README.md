Shortcuts
=========
F11 - Fullscreen toggle
Esc - Goto mainmenu

To compile
==========
Install [dmd](http://dlang.org/download.html) and [dub](http://code.dlang.org/download).
Then just run
> dub
And if you are on windows move the dlls from the release folder to the folder where the exe is.
